# 🎓 EduVault – College Resource Management Portal

## 📌 Overview
EduVault is a web-based platform that helps students easily access and share academic resources like notes, previous year question papers, lectures, and books. It also includes features like cab sharing and blogs to support daily student needs. The platform reduces dependency on multiple apps by providing everything in one place.

---

## 🚀 Features
- 📚 Upload & download notes (with file support)
- 🎓 Senior lectures marketplace
- 📖 Book buying & selling system
- 🚕 Cab/Ride sharing feature
- 📝 Blog section for student guidance
- 📄 Previous Year Papers (PYQ viewer)
- 💬 Basic chatbot assistant
- 📱 Responsive UI design

---

## 🛠️ Tech Stack
**Frontend:**
- HTML
- CSS
- JavaScript

**Backend:**
- Node.js
- Express.js

**Other Tools:**
- Multer (for file uploads)
- JSON (for data storage)
